字典
======================================
字典和列表一样，是python里面非常重要的一个基础数据结构，通俗的理解，字典就像其名一样，一个单词对应一个解释，如'a':'我是小a','yuyu':我是鱼鱼，字典对象里可以包含0个或多个键值对(key-value)

定义一个字典
--------------------------------------
有两种方式可以定义字典::

    d1 = {'name': 'yuyu'}
    d2 = dict(name='yuyu')

    print d1
    print d2

输出为::

    {'name': 'yuyu'}
    {'name': 'yuyu'}

添加元素
--------------------------------------

::

    d1 = {}
    d1['a'] = 1
    print d1

    d2 = {}
    d2.update(b=2)
    print d2
